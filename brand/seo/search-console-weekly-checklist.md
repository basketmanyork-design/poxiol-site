# Google Search Console Weekly Checklist for POXIOL

Run this checklist every week for the first 8-12 weeks after launch.

## 1. Sitemap Status
Check:
- Is `https://poxiol.com/sitemap.xml` submitted?
- Is the sitemap status successful?
- How many URLs were discovered?
- Are there sitemap errors?

## 2. Indexing Status
Check these URLs manually in URL Inspection:
- https://poxiol.com/
- https://poxiol.com/free-mockup/
- https://poxiol.com/about/
- https://poxiol.com/factory/
- https://poxiol.com/oem-odm/
- https://poxiol.com/custom-basketball-uniforms/
- https://poxiol.com/custom-soccer-kits/
- https://poxiol.com/custom-baseball-softball-uniforms/
- https://poxiol.com/custom-running-marathon-wear/
- https://poxiol.com/custom-training-wear/

## 3. Coverage Issues
Look for:
- Crawled - currently not indexed
- Discovered - currently not indexed
- Duplicate without user-selected canonical
- Soft 404
- Not found 404
- Redirect errors

## 4. Search Performance
Track:
- Total clicks
- Total impressions
- Average position
- Queries containing `POXIOL`
- Queries containing `custom teamwear`
- Queries containing `custom basketball uniforms`
- Queries containing `custom soccer kits`
- Queries containing `OEM sportswear`

## 5. Query Intent Review
Group queries into:
- Brand queries
- Product queries
- Supplier/manufacturer queries
- Free mockup queries
- OEM/ODM queries
- Informational queries

## 6. Page Performance
Check which pages get impressions:
- Homepage
- Free Mockup
- Sports category pages
- About
- Factory
- OEM/ODM
- Blog/articles if published

## 7. Action Plan
If a page is not indexed after 2-4 weeks:
- Improve content depth
- Add internal links from homepage/footer
- Add FAQ section
- Add image alt text
- Request indexing manually

## 8. Weekly Notes
Record:
- Date
- Indexed pages count
- Impressions
- Clicks
- Top 10 queries
- Pages needing improvement
