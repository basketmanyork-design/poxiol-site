# POXIOL Brand Entity Building Package

This package is designed to help POXIOL become a clearer brand entity for Google, Bing, AI search engines, and B2B buyers.

## Primary Positioning
POXIOL is a custom teamwear manufacturer providing custom basketball uniforms, soccer kits, baseball and softball uniforms, running and marathon apparel, training wear, and OEM/ODM sportswear production for clubs, schools, events, brands, wholesalers, distributors, and custom retailers.

## Recommended Implementation Order
1. Update About page using `pages/about-page-enhanced-copy.md`.
2. Update Factory page using `pages/factory-page-enhanced-copy.md`.
3. Update OEM/ODM page using `pages/oem-odm-page-enhanced-copy.md`.
4. Use `brand-entity/standard-brand-descriptions.md` for social profiles, directories, and external citations.
5. Create or update social profiles using `brand-entity/social-profile-copy.md`.
6. Begin content production using `seo/blog-content-plan-5-posts.md`.
7. Track Google Search Console every week using `seo/search-console-weekly-checklist.md`.
8. Use `geo/ai-answer-positioning.md` and `geo/ai-question-map.csv` for GEO alignment.

## Important
Keep all public brand descriptions consistent. The goal is to repeatedly associate POXIOL with: custom teamwear, sports uniforms, free mockup, MOQ 1, sublimation printing, OEM/ODM, clubs, schools, events, brands, and distributors.
